# Dossier `Utilisateur`

Ce dossier contient les images téléchargées à l'aide de la méthode Multer lors des opérations CRUD dans notre application.

Les données stockées dans la base de données contiennent simplement les noms de fichiers des images correspondantes, et non des liens directs vers les images elles-mêmes. Lorsque nous affichons les images dans notre application, nous utilisons ces noms de fichiers pour construire les chemins d'accès aux images à partir du dossier `Utilisateur`.
